# Claude Integrations

A central repository showcasing pre-built integrations between Claude and various services using the Model Context Protocol (MCP).

## Overview

This repository contains pre-built integrations for connecting Claude to various third-party services through the Model Context Protocol (MCP). Each integration provides a set of tools and resources that enable Claude to interact with external services.

## Available Integrations

1. **Atlassian**
   - Jira - Issue tracking and project management
   - Confluence - Team collaboration and knowledge management

2. **Zapier** - Connection to thousands of apps and services

3. **Cloudflare** - Web performance and security services

4. **Intercom** - Customer messaging platform

5. **Asana** - Project management and task tracking

6. **Square** - Payment processing and point-of-sale

7. **Sentry** - Application monitoring and error tracking

8. **PayPal** - Online payment processing

9. **Linear** - Software project management

10. **Plaid** - Financial services and banking APIs

## Getting Started

1. Clone this repository:
   ```
   git clone https://github.com/your-username/claudeintegrations.git
   ```

2. Navigate to the specific integration directory you want to use.

3. Follow the setup instructions in the integration's README.md file.
